# Aviator Crash Front-End

## Environment Setup

- `Yarn:     v1.22.21`
- `Node.Js:  v16.14.0`

## Available Scripts

In the project directory, you can run:

## For Project Installation

### `yarn install`

If the above command fails, you can run

### `yarn install --force`


## For Project Start

### `yarn start`

## For Development

### `yarn dev`

## For Production Build

### `yarn build`
